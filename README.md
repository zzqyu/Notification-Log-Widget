 # 은행 계좌 잔고 위젯
위 깃허브 소스를 기반으로 만들었습니다. 

데모로 하나은행의 어플리케이션인 "하나알리미"를 기준으로 만들었습니다. 

은행 계좌에 입송금으로 인해 발생하는 Notification을 Parsing 하여
위젯으로 정리해 표출하는 어플리케이션 입니다.
