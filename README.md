 # 은행 계좌 잔고 위젯
![홍보배너](./banner3.jpg)

위 깃허브 소스를 기반으로 만들었습니다.

사용불가은행: 신한, 부산

테스트완료은행: 하나, 국민, 농협, 카카오, sc제일, 우리
